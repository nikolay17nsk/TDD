Для начала работы скрипт попросить Вас внести по очереди Token, API ключ, язык вывода(en или ru, если нечего не вводить то по дефолту бедет ru).
После чего можно будет вводит адрес(пример - Новосибирск Спартака), после чего выводится пронумерованный список адресов
(если не один из адресов не подходит ввиде 1, после чего появится первый адрес с даннами и можно будет по новой задать более точный адрес),
далее вводите число равнозначное нужному Вам порядковому номеру адреса, затем произойдет вывод выбраного вами ранние адреса с широтой и долготой.
Для выхода нужно ввести exit.
